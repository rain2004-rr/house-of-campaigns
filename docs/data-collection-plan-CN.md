## 二、设备与拍摄规范

### 📱 设备要求

| 项目 | 要求 |
|------|------|
| 分辨率 | 1080P 及以上 (1920×1080) |
| 帧率 | 30fps 或 60fps |
| 色彩 | 彩色，曝光正常，无明显过曝/欠曝 |
| 格式 | MP4 / MOV |
| 时长 | 单条视频 ≥ 10 分钟 |

### 🎥 佩戴方式

**推荐位置**（二选一）：
1. **胸前固定** - 使用胸带/挂绳，相机位于胸口正中
2. **头顶固定** - 使用头带/帽子，相机位于额头正中

**相机方向**：
- 镜头朝前，略微向下倾斜（约 15-30°）
- 确保双手和操作的物体始终在画面内
- 避免剧烈晃动

### ✅ 拍摄角度示意

```
正确示例：
    ┌─────────────┐
    │   [相机]    │  ← 胸前/头顶
    │     ↓       │
    │   👐👐      │  ← 双手可见
    │   [物体]    │  ← 操作对象在画面中央
    └─────────────┘

错误示例：
    ┌─────────────┐
    │   [相机]    │
    │     ↓       │
    │             │  ← 手在画面外 ❌
    │   [物体]    │
    └─────────────┘
```

## 三、质量验收标准

### ✅ 合格视频必须满足

| 检查项 | 标准 |
|--------|------|
| 分辨率 | ≥ 1920×1080 |
| 时长 | ≥ 10 分钟 |
| 画面稳定 | 无明显抖动、模糊 |
| 曝光 | 光线充足，无过暗/过曝 |
| 手部可见 | 双手全程在画面内 |
| 物体可见 | 被操作物体清晰可见 |
| 场景连贯 | 连续拍摄，无频繁中断 |

### ❌ 不合格情况

- 分辨率低于 1080P
- 时长不足 10 分钟
- 手部频繁出画
- 画面过暗/过曝无法辨认
- 视频损坏无法播放

## 六、常见问题 FAQ

### Q1: 可以用手机拍摄吗？
**A**: 可以！确保手机固定在胸前或头顶，使用后置摄像头，分辨率设置为 1080P 或更高。

### Q2: 需要连续拍摄 10 分钟吗？
**A**: 是的，单条视频需要连续拍摄至少 10 分钟，避免频繁暂停/中断。

### Q3: 可以剪辑视频吗？
**A**: 不建议。请提交原始拍摄视频，不要剪辑或后期处理。

### Q4: 光线不好怎么办？
**A**: 请选择白天或光线充足的时间拍摄，必要时开启室内灯光，确保画面清晰。

### Q5: 可以多人一起拍摄吗？
**A**: 可以，但每条视频只能有一个主要操作者（双手出镜者）。

### Q6: 提交后多久能收到激励？
**A**: 任务结束后 3-5 个工作日内，我们会审核视频质量并发放激励。
