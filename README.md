## 📹 Current Campaign: Codatta Home Scene Data Collection

**Goal**: Collect high-quality first-person POV video data of daily home activities.

### Campaign Details

| Item | Description |
|------|-------------|
| **Scenes** | Kitchen cooking, House cleaning, Home tidying |
| **Duration** | Each video ≥ 10 minutes |
| **Quality** | 1080P+, color, normal exposure |
| **Incentive** | $1-2 USD per hour |
| **Deadline** | Weekend (April 20, 2026) |

## 📐 Camera Setup

```
      [Camera]
        ↓ 15-30°
      👐👐
     [Object]
```

**Checklist**:
- [ ] Both hands visible
- [ ] Object clear in frame
- [ ] Stable mount (no shaking)
- [ ] Good lighting
- [ ] 10+ minutes battery
